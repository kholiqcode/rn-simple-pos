## Simple Point Of Sale

SimplePOS adalah aplikasi berbasis mobile yang di buat dengan react native

## Fitur

- List Product
- Add to Cart
- Order
- Order History
- Print Receipt

## Tech Stack

- React Native
- Zustand (State Management)
- React Native MMKV (Storage)
- Shopify Restyle (UI)
- React Native Bluetooth Printer
- i18next (Multi Language)
